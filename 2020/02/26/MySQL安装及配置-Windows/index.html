MySQL安装及配置(Windows)

Posted on 2020-02-26

MySQL安装及配置-Windows（zip版）

  1. 解压mysql目录下创建data和my.ini

  2. 修改配置文件my.ini，按照下方的注释做相应的修改，还有其他参数未写上，官网找咯。

  3. 以管理员身份进入cmd（一定要噢不然权限不够）来到mysql/bin目录下
    执行：mysqld --initialize-insecure --user=mysql

  • 如果想装多几个mysql服务，可以mysqld install 服务名 不然就直接执行：mysqld install
    安装完毕后，执行：net start 服务名 // 如果没有自定义服务名，一般服务名为mysql
    然后进入mysql shell修改root密码（参考下面方法一）

修改MySQL的root密码

方法一（例子为修改root）：
进入mysql shell后，执行：set password for root@localhost = password('newpsd');

方法二（使用mysqldadmin修改密码）：
在bin目录下执行：mysqldadmin -u用户名 -p旧密码 password新密码

方法三（强制修改，在忘记密码时使用）：

  1. 修改本地mysql目录中my.ini文件，在[mysqld]下添加skip-grant-tables

  2. 开启、关闭服务：

  3. 命令行输入mysql -uroot -p，不输密码回车就好。

  4. 执行语句：update user set authentication_string=password("123456") where user="root";然后刷新权限：执行flush privileges;　
  5. 把刚刚修改的my.init那条一句删掉，完成！
